Payroll Data Analysis using SQL

A SQL-based project to design and analyze an employee payroll database.

Features:
- Designed relational database schema for employees, attendance, and salary
- Implemented primary and foreign key constraints
- Performed JOIN operations to combine payroll and attendance data
- Used aggregate functions to calculate total and department-wise payroll
- Identified top earners and employees earning above-average salary

Technologies Used:
- MySQL
- SQL Joins
- Subqueries
- Aggregate Functions
